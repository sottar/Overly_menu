# Overly_menu
Overly_menu is a jQuery plugin for the easiest way for make smart creating global menus 

## Usage

[.zip Download (Mac)](http://www.sottar.net/dist/overly_menu_master.zip)  
~~[.zip Download (Win)]()~~  


    <script>
      $(function() {
        $('#overly_menu').overly_menu();
      });
    </script>

### Options  
 Options: default  
    'speed': 300,  
    'color' : 'rgba(0, 0, 0, 0.8)',  
    'opacity': 0.8,  
    'style': 'list'
